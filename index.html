There are nearly 30'000 molecules that we humans know about, let's
find out more of them!
Normalizing Flows is a generative model that performs non-linear
transformations between probability distributions. It starts with
a base case (i.e. Gaussian, Bernoulli, Pareto) and applies
transformations on the distribution based on a known molecule dataset
(QM9). This transformed distribution can take samples and find new
molecules.
Source code

Institution: Google Summer of Code, 2022.
Language: Python.

Natural Language Processing (NLP) is an key tool for interpretation
and is used in different fields such as chatbots, translators, and
science. BERT transformers are an important part of NLP, as they are
algorithms that learn from the context defined by a user. ESPOL Q&A
Website is a question and answer program where the user can ask
questions about ESPOL University and get the answers through the use
of BERT transformers. In order to be able to respond to the user,
ESPOL Q&A Website gathers data using web scrapping, from official and
reviewed sites, each time a question is being asked. Overall, this
project proves the various uses of BERT transformers, from Q&A
websites to life sciences.

In Ecuador, agro-industries were built under empirical knowledge but
there is a growing opportunity for technology. Data Science and AI are
powerful tools that can be used to computerized methods and optimize
processes.
Essential Oils Optimization is a multivariate optimization that maximizes
yield while minimizing costs. The main structure of this work is to get
the inputs and outputs of the essential oils industry as vectors and
perform a regression on them. Once the regression presents a considerable
result, based on MSE and R^2, the multivariate optimization algorithm
determines the conditions for the facility to work.

Institution: ESPOL and University of Toronto Collaboration, 2021.
Language: Python.
